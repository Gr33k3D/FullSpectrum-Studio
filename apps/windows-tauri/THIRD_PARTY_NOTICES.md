# Third-Party Notices

## Three.js

The Windows/Tauri viewer uses Three.js for WebGL rendering, GLB/GLTF loading,
OBJ loading and orbit camera controls.

- Project: https://threejs.org/
- Source: https://github.com/mrdoob/three.js/
- License: MIT

## Prime3D Reference

Prime3D was reviewed as a design reference for a clean browser-based 3D
preview experience. No Prime3D source code was copied into FullSpectrum Studio.
The FullSpectrum Windows viewer is an original Three.js implementation built
around FullSpectrum's own source/reference workflow, build plate scale overlay
and validation metadata.

- Project: https://primed3d.com/
- Source: https://github.com/3DRev/Primed3D
- Listed license: Apache-2.0
